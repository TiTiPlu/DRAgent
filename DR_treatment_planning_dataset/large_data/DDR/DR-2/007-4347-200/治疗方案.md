1. 疾病诊断：右眼中度非增殖性糖尿病视网膜病变，合并2型糖尿病（病程7年，HbA1c 7.2%）、高血压及糖尿病肾病。
2. 治疗原则：以优化全身系统性疾病控制为核心，延缓糖尿病视网膜病变进展；严密眼科随访，早期识别并干预视网膜病变进展或糖尿病黄斑水肿。
3. 治疗方案：
   · 药物治疗：
      - 全身药物治疗：需与内分泌科及肾内科协作优化。血糖控制目标：HbA1c < 6.5%，空腹血糖 ≤ 6.1 mmol/L，餐后2h血糖 ≤ 7.8 mmol/L。根据eGFR调整二甲双胍（eGFR ≥ 45：足量；30-45：减半；< 30：停用）。优先联用SGLT2i（eGFR ≥ 25-30）或GLP-1 RA；若eGFR < 30，首选GLP-1 RA或胰岛素。逐步替换磺脲类药物。血压目标 < 130/80 mmHg，优先选用ACEI/ARB。启动他汀类药物治疗，考虑非诺贝特（根据eGFR调整剂量）。肾脏保护：SGLT2i/GLP-1 RA联合ACEI/ARB。
      - 眼部药物治疗：暂无。
   · 手术治疗：暂无。
4. 随访计划：眼科每3-6个月全面检查（视力、眼压、裂隙灯、散瞳眼底、OCT、超广角眼底照相）。全身定期监测HbA1c、血压、血脂、UACR、eGFR，与内分泌科及肾内科密切沟通。
5. 患者教育：严格控制血糖、血压、血脂及管理肾脏并发症；健康饮食、戒烟、适度运动；定期自我监测血糖和血压；出现视力突然下降、视物变形、黑影飘动或视野缺损立即就诊；强调按时复诊和遵医嘱。
6. 注意事项：合并肾病时避免或谨慎使用含碘造影剂。联合用药风险管理：SGLT2i与ACEI/ARB联用需监测血压和eGFR；他汀与非诺贝特联用需警惕肌肉毒性并监测CK；未来若行白内障手术，术前需控制视网膜病变及黄斑水肿。
