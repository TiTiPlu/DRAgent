1. 疾病诊断：
右眼：中度非增殖性糖尿病视网膜病变。全身情况：45岁女性，2型糖尿病病史3年，HbA1c 7.0%，血糖控制不佳。

2. 治疗原则：
严格平稳控制血糖、血压、血脂，延缓视网膜病变进展；定期眼部随访，早期干预；目前无需眼部侵入性治疗。

3. 治疗方案：
· 药物治疗
   - 降糖治疗：目标HbA1c < 6.5%，空腹血糖 4.4-7.0 mmol/L，非空腹血糖 < 8.0 mmol/L。若单药不达标，可联合DPP-4抑制剂或SGLT2抑制剂，避免HbA1c骤降（数月内下降超过1.5%-2%）。
   - 血压管理：目标 < 130/80 mmHg，若高于目标，首选ACEI或ARB类药物。
   - 血脂管理：LDL-C目标 < 2.6 mmol/L，若异常，考虑他汀类药物治疗。
· 手术治疗
   暂无。

4. 随访计划：
眼科：每3-6个月散瞳眼底检查；若出现视力下降、视物模糊、视野异常、飞蚊或闪光感，立即复诊，考虑OCT检查。内分泌：定期复查HbA1c、血糖、血压、血脂、UACR及eGFR。

5. 患者教育：
告知糖尿病对眼睛的潜在危害及早期筛查的重要性；强调控制血糖、血压、血脂是保护视力的基石；指导饮食控制（食盐<5g/天，限制饱和脂肪酸）、适度运动（每周至少150分钟中等强度有氧运动）、保持理想体重（BMI < 24.0 kg/m²）、戒烟限酒；教育识别视力异常症状；强调长期规范治疗和定期随访。

6. 注意事项：
避免过度治疗，目前不需激光或注射；需内分泌科与眼科共同管理；降糖应平稳缓慢，预防视网膜病变早期恶化；警惕低血糖；即使无症状也需定期复查；关注黄斑区情况，必要时OCT检查；定期检查肾功能及血脂；就诊时主动告知用药情况。
