1. 疾病诊断：左眼中度非增殖性糖尿病视网膜病变，伴中心凹受累型糖尿病黄斑水肿；2型糖尿病（病程4年，血糖控制不佳，HbA1c 8.3%），合并糖尿病肾病。

2. 治疗原则：眼部优先控制黄斑水肿，避免过度治疗；全身与内分泌科协作，平稳控制血糖、血压、血脂，保护肾功能，避免血糖剧烈下降。

3. 治疗方案：
   · 药物治疗：眼科：立即启动左眼玻璃体腔内注射抗VEGF药物（如雷珠单抗等），初始每月一次，连续至少3次，后续按需或延伸治疗。全身：继续口服二甲双胍（根据eGFR调整剂量）；加用SGLT-2抑制剂或GLP-1受体激动剂（优先考虑，注意肾功能）；暂停或避免使用噻唑烷二酮类药物；若1-2个月后HbA1c仍>8.0%，考虑启动基础胰岛素；使用ACEI/ARB控制血压（<130/80 mmHg），他汀类药物调脂。
   · 手术治疗：目前无全视网膜激光光凝需求。若抗VEGF治疗后黄斑水肿持续，可考虑辅助性微脉冲激光或局灶格栅样激光，但初始不推荐。

4. 随访计划：眼科：抗VEGF注射期每1-2个月复查视力、眼压、OCT；病情稳定后逐渐延长至2-4个月。全身：每1-3个月复查HbA1c、空腹血糖、肾功能、血脂、血压，调整治疗方案。

5. 患者教育：解释糖尿病视网膜病变及黄斑水肿病因和治疗必要性；强调平稳控制血糖、血压、血脂的重要性，告知血糖快速下降可能致视力波动；说明抗VEGF治疗作用、风险和按时复诊必要性；指导健康生活方式和用药依从性；告知注射并发症及SGLT-2抑制剂副作用，指导自我监测视力变化，异常及时就医。

6. 注意事项：内分泌科会诊制定个体化方案，初期HbA1c目标7.5%-8.0%，稳定后逐步向7.0%靠拢；注射前排除感染，注射后观察并发症；合并肾病时谨慎用药；警惕病变进展至增殖期，长期随访，必要时重新评估全视网膜光凝；密切监测肾功能，尤其使用SGLT-2抑制剂期间。
