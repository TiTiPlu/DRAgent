1. 疾病诊断：
右眼：中度非增殖性糖尿病视网膜病变（NPDR），伴有视力下降和黄斑颗粒样改变，提示糖尿病性黄斑水肿（DME）可能。
全身：2型糖尿病（血糖控制不佳，HbA1c 8.4%），高脂血症（间断服药）。
其他：右眼视网膜神经纤维髓鞘样改变。

2. 治疗原则：
严格控制全身性疾病（糖尿病、高脂血症），设定明确控制目标。
稳定并改善视力，预防视网膜病变和黄斑水肿进展。
密切监测眼部及全身情况。

3. 治疗方案：
· 药物治疗
全身治疗：
血糖控制目标：短期（3-6个月）HbA1c < 7.0%，长期争取< 6.5%；空腹血糖4.4-7.0 mmol/L，餐后2小时血糖< 10.0 mmol/L。
血脂控制目标：LDL-C < 1.8 mmol/L，可考虑< 1.4 mmol/L。
药物调整及强化：生活方式干预（糖尿病自我管理教育、医学营养治疗、运动）；降糖药物在二甲双胍基础上加用SGLT2抑制剂或GLP-1受体激动剂，必要时起始基础胰岛素；调脂药物优化他汀类药物（如阿托伐他汀或瑞舒伐他汀），必要时联用依折麦布或PCSK9抑制剂。
眼部治疗：抗血管内皮生长因子（抗VEGF）药物玻璃体腔注射（如雷珠单抗或阿柏西普）；初始负荷治疗每月一次连续注射3次，后续按需（PRN）或维持（T&E）方案调整。
· 手术治疗
暂无

4. 随访计划：
眼科随访：抗VEGF注射后1个月内复查；负荷治疗期间每月随访（视力、眼压、眼前段及眼底检查、黄斑OCT）；维持期每1-3个月随访。
全身随访：每3-6个月随访内分泌科医生，监测HbA1c、空腹血糖、餐后2小时血糖、血脂谱、血压、尿微量白蛋白；每年评估肾功能和心血管风险。

5. 患者教育：
强调严格控制血糖、血脂、血压，明确目标（HbA1c < 7.0%，LDL-C < 1.8 mmol/L）以延缓视网膜病变进展。
指导患者理解抗VEGF治疗的目的、过程、效果及风险。
进行糖尿病自我管理教育，解决不规律用药原因，强化医学营养治疗和运动计划，教育低血糖识别与处理。
告知日常观察视力变化，如有急性视力下降、视物变形或眼前黑影立即就医。
强调定期眼科和全身随访，不可自行中断治疗。
建议戒烟，保持健康生活方式。

6. 注意事项：
抗VEGF注射后密切观察眼部感染迹象（眼红、眼痛、视力急剧下降）或其他并发症。
关注强化全身治疗风险，特别是低血糖识别与处理，以及新型降糖药副作用（胃肠道不适、泌尿生殖道感染）。
关注患者全身情况，避免因眼科治疗忽视糖尿病系统管理。
治疗方案调整需基于每次随访检查结果和患者反应。
需排除其他导致视力下降的原因。
