1. 疾病诊断：右眼中度非增殖性糖尿病视网膜病变，伴黄斑区散在硬性沉积；2型糖尿病8年（血糖控制不佳，HbA1c 7.8%）；高血压病史5年（控制不佳）。

2. 治疗原则：严格控制全身代谢指标，目标为HbA1c < 7.0%（争取≤6.5%），血压 < 130/80 mmHg，LDL-C < 2.6 mmol/L；减轻黄斑水肿，稳定视力，延缓视网膜病变进展。

3. 治疗方案：
   · 药物治疗：全身管理：血糖管理目标HbA1c < 7.0%，争取≤6.5%，建议加用SGLT2i或GLP-1RA；血压管理目标< 130/80 mmHg，建议加用ACEI或ARB；血脂管理目标LDL-C < 2.6 mmol/L，必要时启动他汀类。眼部治疗：启动玻璃体腔内抗VEGF药物注射（如雷珠单抗、阿柏西普），连续每月注射3次，后续按PRN或T&E方案调整。
   · 手术治疗：暂无

4. 随访计划：全身随访：定期至内分泌科和心内科复诊；HbA1c每3-6个月监测一次；加强自我血糖监测；每日家庭血压监测；血脂每6-12个月监测一次；定期筛查肾功能及尿蛋白。眼部随访：抗VEGF负荷治疗期间每月复查视力、眼压、眼前段和玻璃体状况及OCT；负荷治疗结束后每1-3个月复查一次。

5. 患者教育：解释疾病进展、治疗目标和预期效果；强调严格控制血糖、血压、血脂是基础，并告知全身用药的协同保护作用；告知抗VEGF治疗的必要性、益处和风险（如眼内炎、眼压升高）；指导识别视力变化预警信号，按时复诊；强调长期管理需要高度依从性。

6. 注意事项：全身疾病控制不佳将直接影响眼部治疗效果；需密切监测抗VEGF治疗并发症；患者需保持良好生活习惯（均衡饮食、适度运动）；关注对侧眼视网膜健康，定期双眼筛查。
