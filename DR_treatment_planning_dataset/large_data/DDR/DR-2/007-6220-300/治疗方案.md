1. 疾病诊断：右眼中度非增殖性糖尿病视网膜病变；2型糖尿病控制欠佳（HbA1c 8.2%）；高血压病。右眼黄斑区未见明显黄斑水肿。
2. 治疗原则：优化全身代谢控制，延缓糖尿病视网膜病变进展，定期眼科监测，个体化治疗，避免过度治疗和并发症，多学科协作。
3. 治疗方案：
   · 药物治疗：眼部药物治疗：无。全身药物治疗：糖尿病管理：与内分泌科协作制定个体化降糖方案；首要步骤：评估目前口服降糖药和降压药及肾功能、肝功能等；血糖控制目标：短期HbA1c < 7.0%~7.5%，空腹血糖 ≤ 7.0 mmol/L，餐后2小时血糖 ≤ 10.0 mmol/L，HbA1c下降速度每3个月不超1%-1.5%；药物策略：初期联合治疗，二甲双胍为基础，联合SGLT2抑制剂（如达格列净10mg每日一次）或GLP-1受体激动剂（如利拉鲁肽从0.6mg每日一次起始滴定）；胰岛素治疗为备选。高血压管理：与心内科协作调整降压药，目标血压 < 130/80 mmHg，优先使用RAS抑制剂。血脂管理：启动或调整他汀类药物，目标LDL-C根据风险分层 < 1.8 mmol/L或 < 2.6 mmol/L。
   · 手术治疗：暂无。
4. 随访计划：眼科随访：3个月后复查右眼眼底，包括视力、眼压、OCT等。全身检查随访：每1-3个月复查HbA1c、血糖、血压、血脂、肾功能等。
5. 患者教育：告知严格控制血糖、血压、血脂的重要性；定期眼科检查；出现视力突然下降等症状立即就诊；保持健康生活方式；药物使用教育包括低血糖识别处理、SGLT2i警惕泌尿生殖道感染和脱水、GLP-1RA警惕胃肠道不适；强调规律监测血糖、血压、体重。
6. 注意事项：严格遵医嘱，勿自行调整剂量；HbA1c下降速度不宜过快；多学科评估药物相互作用；避免可能引起眼部不适的活动；药物特异性注意事项：二甲双胍需排除禁忌证（eGFR<30等），碘造影剂检查前后暂停；SGLT2i需评估肾功能（eGFR≥25-30），警惕感染和脱水；GLP-1RA排除甲状腺髓样癌史，关注胃肠道反应；他汀类药物监测肝功能和肌酶。
