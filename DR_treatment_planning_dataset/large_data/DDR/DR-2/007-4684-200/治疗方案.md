1. 疾病诊断：右眼中度非增殖性糖尿病视网膜病变；右眼视网膜激光光凝术后；右眼轻度晶状体混浊；2型糖尿病（HbA1c 6.8%）。患者为54岁男性，糖尿病病程11年。

2. 治疗原则：严格强化血糖控制以延缓糖尿病视网膜病变进展；维持和保护现有视力；密切监测黄斑水肿及视网膜病变进展；预防低血糖发生。

3. 治疗方案：
   · 药物治疗：内分泌管理：严格控制HbA1c < 6.5%，空腹血糖 ≤ 6.1 mmol/L，餐后2小时血糖 ≤ 7.8 mmol/L。药物调整：建议加用胰高血糖素样肽-1受体激动剂；若不耐受，可考虑SGLT-2抑制剂或DPP-4抑制剂；若血糖仍不达标，启动胰岛素治疗。其他全身管理：血压<130/80 mmHg；管理血脂异常。眼部药物治疗：暂无。
   · 手术治疗：当前无视网膜激光光凝治疗指征；右眼轻度晶状体混浊无需手术干预，当视力显著下降影响日常生活时考虑白内障摘除术。

4. 随访计划：眼科随访：6个月后复查，包括散瞳眼底检查及黄斑区光学相干断层扫描；持续监测晶状体混浊进展。全身系统随访：立即调整降糖方案并密切随访，确保血糖稳定达标；定期随访内分泌科或全科医生，维持血压和血脂理想水平。

5. 患者教育：强调严格控制血糖（HbA1c < 6.5%）、血压、血脂的重要性；告知眼部症状（视力突然下降、眼前黑影增多、闪光感、视物变形）需立即就医；强调定期眼科和全身检查的重要性；告知右眼轻度白内障可能进展，未来需手术；鼓励健康生活方式（均衡饮食、适度运动）。

6. 注意事项：避免可能导致眼压升高的剧烈活动；户外活动时佩戴防紫外线眼镜；严格遵医嘱使用降糖药物，定期监测血糖，避免波动过大，警惕低血糖并学会处理。
