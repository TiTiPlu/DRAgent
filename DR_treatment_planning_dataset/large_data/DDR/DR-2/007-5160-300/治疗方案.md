1. 疾病诊断：右眼中度非增殖性糖尿病视网膜病变伴糖尿病黄斑水肿，黄斑区轻度增厚倾向，视力0.6；左眼视力0.8，无明显活动性病变；右眼轻度核性白内障。合并2型糖尿病5年，血糖控制不佳（HbA1c 7.8%），高血压3年。

2. 治疗原则：改善右眼视力，控制糖尿病黄斑水肿进展，稳定视网膜病变，预防疾病发展；加强全身系统性疾病（糖尿病和高血压）的严格控制。

3. 治疗方案：
   · 药物治疗：右眼玻璃体腔内注射抗VEGF药物（如雷珠单抗、阿柏西普、法瑞西单抗等），负荷剂量治疗（连续3次，每月1次），后续按需或维持方案（随访间隔4-8周）。全身治疗：转诊内分泌科，优化2型糖尿病治疗；血糖控制目标HbA1c < 6.5%，空腹血糖 ≤ 6.1 mmol/L，餐后2小时血糖 ≤ 7.8 mmol/L；药物路径：基础用二甲双胍，若单药不达标，联合DPP-4抑制剂（如西格列汀，肾功能正常者100mg qd，中度肾功能不全者50mg qd）或其他机制药物；若口服药无效，尽早启动胰岛素治疗（如甘精胰岛素或预混胰岛素）。血压控制：监测并严格控制血压。
   · 手术治疗：暂无。目前无需激光光凝手术；白内障程度轻微，优先处理糖尿病黄斑水肿。

4. 随访计划：眼部随访：负荷剂量期每月1次，进行眼部检查、视力评估、眼压测量及OCT检查；维持治疗期每4-8周1次，定期眼底照相。全身随访：定期至内分泌科复查血糖、糖化血红蛋白；定期监测血压。

5. 患者教育：强调严格控制血糖（遵医嘱规律服药、饮食控制、适度运动）和血压，具体目标HbA1c < 6.5%，空腹血糖 ≤ 6.1 mmol/L，餐后2小时血糖 ≤ 7.8 mmol/L；针对用药不规律史，制定服药提醒计划。详细解释右眼糖尿病黄斑水肿诊断及抗VEGF治疗的必要性、流程（负荷剂量和维持治疗）、效果及风险。告知定期眼科随访的重要性，识别视力急性下降、视物变形等早期症状，及时就诊。说明抗VEGF治疗后可能的并发症（如眼部不适、红肿、感染风险）及应对措施。

6. 注意事项：抗VEGF注射前需全面眼部检查，排除感染；注射后遵医嘱使用抗生素滴眼液预防感染；密切监测眼压；白内障手术应在糖尿病黄斑水肿稳定并控制后考虑；全身系统性疾病良好控制对眼部病变长期稳定至关重要。
