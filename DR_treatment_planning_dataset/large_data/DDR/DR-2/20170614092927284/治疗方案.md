1. 疾病诊断：左眼中度非增殖性糖尿病视网膜病变，伴轻度糖尿病黄斑水肿；双眼晶状体轻度核硬化；2型糖尿病10年，糖化血红蛋白7.1%。

2. 治疗原则：严格控制全身性疾病，维持代谢稳定；积极治疗左眼糖尿病黄斑水肿，改善及维持视力；预防糖尿病视网膜病变进一步发展；黄斑水肿稳定前不考虑其他眼内手术。

3. 治疗方案：
   · 药物治疗：左眼玻璃体腔内注射抗VEGF药物，初期连续3个月每月一次强化负荷治疗，后根据反应调整为按需治疗或维持-延长方案。全身管理：转诊内分泌科优化降糖方案，优先考虑SGLT2抑制剂或GLP-1受体激动剂；启动或调整降压及调脂治疗，目标HbA1c < 7.0%，血压 < 130/80 mmHg，LDL-C < 1.8 mmol/L或更低。
   · 手术治疗：暂无

4. 随访计划：按抗VEGF治疗周期定期复查，评估黄斑水肿、视力及眼底病变进展；每次复查行眼底检查和光学相干断层扫描。

5. 患者教育：解释糖尿病视网膜病变及黄斑水肿病程、治疗方案及预期效果；强调严格控制血糖、血压、血脂的重要性（HbA1c < 7.0%，血压 < 130/80 mmHg，LDL-C < 1.8 mmol/L或更低）；指导规律自我血糖监测；识别视力下降、视物变形等症状；告知注射后反应及注意事项；按时随访，积极配合治疗。

6. 注意事项：抗VEGF注射后注意眼部卫生，避免感染，短期内避免剧烈活动；持续与内分泌科协作管理全身糖尿病，定期复查糖化血红蛋白，优化全身用药；加强低血糖风险教育及监测；左眼黄斑水肿稳定前暂不考虑白内障手术；右眼定期眼底检查。
