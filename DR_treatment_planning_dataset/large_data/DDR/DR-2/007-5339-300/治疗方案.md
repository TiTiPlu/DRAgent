1. 疾病诊断：右眼中度非增殖性糖尿病视网膜病变，双眼无临床显著性糖尿病黄斑水肿，2型糖尿病合并高血压病。

2. 治疗原则：严格控制血糖、血压及血脂，目标：HbA1c < 6.5%，空腹血糖 ≤ 6.1 mmol/L，餐后2小时血糖 ≤ 7.8 mmol/L，血压 < 130/80 mmHg，LDL-C < 2.6 mmol/L。密切观察视网膜病变进展。

3. 治疗方案：
   · 药物治疗：眼部无需特异性药物。全身治疗：糖尿病管理，以二甲双胍为基础，若单药不达标需在1-3个月内联合SGLT-2抑制剂、GLP-1受体激动剂等；高血压管理，优化降压方案至目标值；血脂管理，降脂治疗使LDL-C < 2.6 mmol/L。警惕药物相互作用。
   · 手术治疗：暂无。

4. 随访计划：眼科每6个月全面检查（视力、眼压、散瞳眼底、OCT），出现新发症状立即复诊。全身指标定期监测血糖、HbA1c、血压、血脂、肾功能，调整药物时增加血糖监测频率。

5. 患者教育：解释视网膜病变分期及进展风险，强调严格控制血糖、血压、血脂目标值，指导识别眼部症状并及时就医，鼓励健康生活方式，强调按时复查。

6. 注意事项：严格遵守全身治疗方案，避免剧烈运动及屏气，注意用眼卫生，所有药物调整需在医生指导下进行，并告知所有用药史。
