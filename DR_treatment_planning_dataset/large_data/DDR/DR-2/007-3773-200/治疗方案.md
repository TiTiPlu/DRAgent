1. 疾病诊断：右眼中度非增殖性糖尿病视网膜病变，伴中心凹受累的糖尿病性黄斑水肿；2型糖尿病6年，血糖控制不佳。
2. 治疗原则：严格控制全身危险因素，积极进行眼部局部治疗以消除黄斑水肿，改善视力，防止视网膜病变进展。
3. 治疗方案：
   · 药物治疗：眼部局部治疗：右眼玻璃体腔内注射抗VEGF药物，初始每月注射3次，后续按需或维持治疗。全身治疗：与内分泌科协作，制定个体化血糖控制方案。初期目标HbA1c < 7.0%，空腹血糖 ≤ 7.0 mmol/L，餐后2小时血糖 ≤ 10.0 mmol/L；长期目标HbA1c < 6.5%，空腹血糖 ≤ 6.1 mmol/L，餐后2小时血糖 ≤ 7.8 mmol/L。以二甲双胍为基础，优先联合SGLT-2抑制剂或DPP-4抑制剂，必要时启动胰岛素治疗。禁止使用噻唑烷二酮类。同时严格控制血压、血脂。
   · 手术治疗：暂无
4. 随访计划：眼部随访：首次抗VEGF注射后每月复查视力、眼压及OCT。全身随访：初期频繁监测血糖，HbA1c每2-3个月一次，定期监测血压、血脂及心血管风险。
5. 患者教育：告知疾病性质、治疗必要性及预期效果；强调严格控制血糖、血压、血脂的重要性；指导遵医嘱用药、规律饮食、适度运动；告知抗VEGF注射风险及注意事项。
6. 注意事项：抗VEGF注射后观察眼压升高、眼内感染等并发症；遵医嘱定期复查，不可中断；强调全身控制对眼底病变预后的关键作用及眼科与内分泌科协作的重要性。
