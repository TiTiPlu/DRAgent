1. 疾病诊断：
右眼轻度非增殖性糖尿病视网膜病变（NPDR），无黄斑水肿及视网膜新生血管；2型糖尿病（病程11年，HbA1c 8.3%控制欠佳）；高血压；糖尿病肾病；右眼轻度老年性晶状体混浊。

2. 治疗原则：
严格控制血糖、血压、血脂，延缓视网膜病变进展，避免过度治疗及低血糖风险。当前无需眼部介入性治疗，重点为全身管理和定期眼科随访。

3. 治疗方案：
   · 药物治疗
     眼部药物：暂无。
     全身药物：
     血糖控制：与内分泌科协作优化降糖方案。目标HbA1c < 8.0%，慎防低血糖；仅在无低血糖风险且依从性佳时可尝试 < 7.5%。启动SGLT2i或GLP-1RA前需完善肾功能评估（血肌酐、eGFR、尿白蛋白肌酐比值）。基于肾功能分层选药：eGFR < 30禁用二甲双胍和大多数SGLT2i；优先DPP-4抑制剂或基础胰岛素。30 ≤ eGFR < 45时，二甲双胍需减量或停用；SGLT2i需评估获益风险比。eGFR ≥ 45方可优先考虑SGLT2i或GLP-1RA。严格遵循药品说明书，初期加强血糖监测防低血糖。
     血压控制：与心内科或肾内科协作，目标<130/80 mmHg，优先选用ACEI或ARB类药物。
     血脂管理：必要时使用他汀类药物。
   · 手术治疗：暂无。

4. 随访计划：
每6个月进行一次眼科复查（视力、眼压、眼前段检查、扩瞳眼底检查、黄斑区OCT）。定期监测血糖、血压、血脂、肾功能（尿白蛋白肌酐比值、肾小球滤过率）。

5. 患者教育：
告知糖尿病视网膜病变与血糖、血压、血脂控制密切相关，是失明主因之一。强调严格控制全身指标，遵医嘱用药，避免低血糖为最高优先级。建议均衡饮食、适度运动、戒烟限酒。出现视力突然下降、视物模糊、变形、黑影、闪光感等症状应立即就医。配合内分泌科、心肾内科及眼科综合治疗。

6. 注意事项：
避免剧烈运动、屏气等导致眼压升高或视网膜出血的活动。散瞳后短时视物模糊，不宜驾车或操作精密机械。糖尿病肾病患者注意控制蛋白质摄入，警惕药物对肾功能影响。即使视力良好也需定期检查。内分泌科制定处方时，防低血糖优先于强控糖，并根据实时肾功能精准选药。
