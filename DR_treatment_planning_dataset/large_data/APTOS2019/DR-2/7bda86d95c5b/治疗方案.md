1. 疾病诊断：
左眼中度非增殖性糖尿病视网膜病变合并糖尿病性黄斑水肿。患者糖尿病病程9年，近期糖化血红蛋白6.8%，存在糖尿病周围神经病变。

2. 治疗原则：
积极控制全身危险因素，稳定血糖、血压、血脂，规范化管理糖尿病并发症，保护视力，延缓疾病进展。针对左眼糖尿病性黄斑水肿进行药物治疗，改善视力，减轻黄斑水肿。定期随访，监测疾病进展。

3. 治疗方案：
   · 药物治疗
       眼部治疗：左眼玻璃体腔内注射抗血管内皮生长因子药物（雷珠单抗、阿柏西普或康柏西普）。初始每月一次，连续3次作为负荷剂量。
       全身药物治疗：
          血糖控制：目标HbA1c < 7.0%，加强动态血糖监测。若肾功能正常且无禁忌症，建议加用SGLT-2抑制剂（如达格列净10mg每日一次或恩格列净10mg每日一次）。若不适用的，考虑GLP-1受体激动剂。
          血压控制：目标血压 < 130/80 mmHg。若血压未达标或存在微量白蛋白尿，立即启用血管紧张素转换酶抑制剂或血管紧张素II受体拮抗剂类药物。
          血脂控制：立即启动高强度他汀治疗（如阿托伐他汀20mg每晚一次），目标低密度脂蛋白胆固醇 < 1.4 mmol/L。若甘油三酯显著升高，可考虑加用非诺贝特。
          神经病变管理：常规启动神经营养及抗氧化治疗（甲钴胺0.5mg每日三次+α-硫辛酸0.6g每日两次）。若伴明显神经痛，可加用普瑞巴林或加巴喷丁。
   · 手术治疗：暂无

4. 随访计划：
眼科随访：每次注射后1个月复查（最佳矫正视力、眼压、裂隙灯、散瞳眼底检查及光学相干断层扫描）。完成负荷剂量后，根据结果调整后续注射间隔。若无注射，每2-3个月进行一次全面眼部检查。进行左眼荧光素眼底血管造影。
全身随访：每3个月复查糖化血红蛋白、血压；每3-6个月复查血脂、肝肾功能、尿常规及尿微量白蛋白/肌酐比值。定期评估糖尿病周围神经病变。

5. 患者教育：
告知患者糖尿病性黄斑水肿是视力下降主因，抗血管内皮生长因子治疗最有效。强调治疗为长期过程，需多次注射和定期随访，切勿中断。指导严格控制血糖、血压、血脂（目标值HbA1c < 7.0%、血压 < 130/80 mmHg、低密度脂蛋白胆固醇 < 1.4 mmol/L），解释相关药物的保护作用。强调规律运动、健康饮食、戒烟限酒。教育自我监测视力，出现急剧下降、黑影、视野缺损等立即就诊。告知注射风险及术后注意事项。告知神经病变管理的重要性。

6. 注意事项：
注射前确保无眼部感染或炎症。密切监测注射后眼压，必要时降眼压。嘱咐注射后避免揉眼、脏水入眼，预防感染。建议立即至内分泌科评估全身情况，优化全身治疗方案，特别筛查早期糖尿病肾病并评估肾小球滤过率以指导SGLT-2抑制剂使用。告知患者未来可能需要评估白内障进展。
