1. 疾病诊断：右眼中度非增殖性糖尿病视网膜病变，合并糖尿病性黄斑水肿（中心凹受累），右眼黄斑裂孔术后，眼底激光治疗后状态。患者有2型糖尿病病史10年，HbA1c 7.2%。

2. 治疗原则：严格控制全身代谢指标（尤其是血糖），减轻黄斑水肿，改善视力，稳定视网膜病变，预防视力进一步下降。

3. 治疗方案：
   · 药物治疗：
     针对糖尿病性黄斑水肿：右眼玻璃体腔内注射抗血管内皮生长因子药物。
     针对全身疾病：控糖目标HbA1c < 6.5%，空腹血糖 ≤ 6.1 mmol/L，餐后2小时血糖 ≤ 7.8 mmol/L。降糖策略：转诊内分泌科，优先使用二甲双胍为基础，加用SGLT-2抑制剂或GLP-1受体激动剂，备选DPP-4抑制剂；若血糖仍不达标，启动胰岛素强化治疗。药物排查与监测：排查噻嗪类利尿剂、糖皮质激素等升糖药物，加强血糖监测频率。
   · 手术治疗：暂无

4. 随访计划：玻璃体腔注射抗VEGF药物后，下次注射前进行眼科检查（视力、眼压、眼底镜、OCT）。长期每3个月进行一次眼科检查，同时配合内分泌科监测血糖、糖化血红蛋白、血压、血脂。

5. 患者教育：强调严格控制血糖（目标HbA1c < 6.5%、空腹血糖 ≤ 6.1 mmol/L、餐后2小时血糖 ≤ 7.8 mmol/L）、血压、血脂的重要性，指导定期监测血糖。告知抗VEGF治疗的预期效果、副作用及并发症。若出现视力下降、视物变形、眼红、眼痛等立即就医。强调与内分泌科配合，可能需使用新型降糖药物，并告知医生所有正在服用的药物。

6. 注意事项：治疗中密切监测眼压变化。长期关注糖尿病病程，预防其他并发症。抗VEGF治疗期间避免剧烈运动，保持眼部清洁。与内分泌科协作管理全身情况，特别关注可能引起血糖波动的合并用药（如噻嗪类利尿剂、糖皮质激素等），必要时调整或加强监测。
