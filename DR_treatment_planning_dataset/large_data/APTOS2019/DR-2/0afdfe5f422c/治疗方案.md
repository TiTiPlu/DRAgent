1. 疾病诊断：右眼黄斑裂孔术后改变；右眼中度非增殖性糖尿病视网膜病变（NPDR）；2型糖尿病（病程10年，HbA1c 6.6%）；高血压。
2. 治疗原则：严格控制血糖、血压、血脂，延缓糖尿病视网膜病变进展；定期眼科随访，监测视网膜及黄斑区情况；当前以观察和全身管理为主，暂不局部眼部干预。
3. 治疗方案：
   · 药物治疗：全身治疗：继续口服降糖药和降压药。目标：HbA1c < 6.5%（空腹血糖 ≤ 6.1 mmol/L，餐后2小时血糖 ≤ 7.8 mmol/L），血压 < 130/80 mmHg。优先通过生活方式干预（饮食微调、增加运动）进一步降低HbA1c；若1-2个月后复查不达标，由内分泌科评估调整降糖方案（如SGLT-2抑制剂或GLP-1受体激动剂）。眼部药物：暂无。
   · 手术治疗：暂无。
4. 随访计划：6个月后眼科复查；项目包括矫正视力、眼压、裂隙灯、前置镜或三面镜眼底检查，每次随访必须行OCT评估黄斑区结构及排除水肿，必要时行眼底广角照相；全身定期检查HbA1c、血压、血脂、肾功能。
5. 患者教育：告知糖尿病视网膜病变进展风险及全身疾病控制重要性；指导正确监测血糖、血压，按时用药，注意避免低血糖；教育识别视力下降、视物变形、黑影、视野缺损等症状，出现需立即就诊；强调定期眼科随访的必要性。
6. 注意事项：严格遵循内分泌科和心血管内科医嘱，在避免低血糖前提下控制血糖（HbA1c < 6.5%）及血压、血脂；避免剧烈运动和屏气动作；保持规律作息、健康饮食；严禁吸烟饮酒；佩戴合适矫正眼镜；避免自行使用眼药水，不适及时就医。
