1. 疾病诊断：左眼增殖性糖尿病视网膜病变（活动期），伴糖尿病性黄斑水肿；既往左眼全视网膜光凝术后；全身2型糖尿病史10年，血糖控制欠佳，合并糖尿病肾病。
2. 治疗原则：控制活动性增殖病变，消除黄斑水肿，稳定视力，预防视力进一步下降。个体化强化全身血糖控制，延缓糖尿病并发症进展，保障患者整体健康。
3. 治疗方案：
   · 药物治疗：左眼玻璃体腔内注射抗血管内皮生长因子药物（康柏西普或雷珠单抗）。全身降糖治疗调整：与内分泌科协作，个体化血糖控制目标（HbA1c < 7.5% 或 < 8.0%，空腹血糖 ≤ 7.0-8.0 mmol/L，餐后2小时血糖 ≤ 10.0 mmol/L），优先考虑二甲双胍（根据eGFR调整剂量）联合SGLT-2抑制剂或GLP-1受体激动剂，慎用或避免磺脲类药物，胰岛素使用长效类似物。肾功能监测与保护：与肾内科协作。
   · 手术治疗：左眼局部补充性全视网膜光凝，氩激光绿色（532 nm），光斑大小200-300 μm，功率150-300 mW，曝光时长0.2-0.5秒，可分次完成，首次重点治疗中周部局灶性细小异常血管改变区域。避让黄斑区（中心凹500 μm范围）、视盘（盘缘500 μm范围）及粗大血管，不重复光凝已反应良好的激光瘢痕区域。
4. 随访计划：眼科随访：抗VEGF首次注射后1个月内复查视力、眼压、眼前节、眼底及OCT，此后每1-3个月定期随访。全身随访：定期检查血糖、血压、血脂及肾功能，与内分泌科、肾内科医生沟通。
5. 患者教育：强调个体化控制血糖、血压、血脂的重要性；告知玻璃体腔内注射和激光治疗的预期效果、风险及并发症；指导识别眼部异常症状（如视力骤降、黑影增多、视野缺损）并及时就诊；强调按时用药和定期复查。
6. 注意事项：密切监测眼压变化，注意新生血管性青光眼风险；糖尿病肾病患者避免使用肾毒性或易致低血糖药物，并根据肾功能调整用药；避免低血糖事件，教会患者识别症状并处理；激光治疗后可能出现短暂性视力模糊，嘱患者注意休息，避免剧烈运动。
